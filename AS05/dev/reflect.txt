In this assignment, I learned more about member/helper/friend functions and operator overloading in C++. Before doing this assignment, I forgot the difference between member functions, non-member helper functions, and friend functions because I took a long break in reading week. After implementing the Mark class, I learned that friend functions can access private members directly, and they should be declared inside the class, but the function definition should be written outside the class. However, helper functions should be declared and written both outside the class.
The most difficult part for me was the input operator as the program must reject invalid input such as trailing characters like 67abc or 67.8. I also practiced using ostream formatting, such as width, precision, fill characters, and left or right alignment.
