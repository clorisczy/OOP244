This assignment made me understand the usage of dynamic memory allocation in C++ more clearly. In the past, my understanding of pointer was vague. It was easy to think that declaring a pointer means that something can be used, but this time I found that this is not the case. For example, Samples* s is just a pointer. If you really want to create an object, you still need to use new Samples. I also learned that new, delete, new[] and delete[] should be used accordingly, otherwise it is easy to cause memory leak.
